# US: Create Task

## User Story

**As a** registered user  
**I want** to create a task  
**So that** it appears on my homepage

## Acceptance Criteria

### AC1: Task Created Successfully
- **Given** the user fills the title input with "Help with oranges"
- **Given** the user fills the message input with "My message"
- **Given** the user fills the organisation input with "My organisation"
- **Given** the user fills the contact information input with "06 06 76 87 98"
- **Given** the user fills the date ranges input with a start date smaller than an end date 
- **When** the user submits the form  
- **Then** the task is created, the user is redirected to the task page
- **Then** the user is redirected to the task page
- **Then** a success toast message appears stating 'Task created'
