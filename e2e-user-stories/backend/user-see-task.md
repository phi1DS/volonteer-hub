# US: See Task

## User Story

**As a** registered user  
**I want** to see my created tasks  
**So that** I can verify they exist and track them

## Acceptance Criteria

### AC1: Task shows up on task page
- **Given** a task with title "Help with cleaning dishes" has been created previously  
- **When** the user navigates to the task page  
- **Then** the created task is displayed on the page
