# US: Close Task

## User Story

**As a** registered user  
**I want** to mark a task as completed  
**So that** I can track which tasks are done and keep my task list organized

## Acceptance Criteria

### AC1: Successfully close task
- **Given** a task with title "Help with furniture" exists for the logged-in user  
- **When** the user marks the task as completed  
- **Then** the task gets removed from the ongoing task list  
- **And** a success toast message appears
